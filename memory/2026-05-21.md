# 2026-05-21

## Summary
- Fixed a bug in `scripts/run-runtime-governance.sh` where empty array expansions of `smoke_overrides` caused "unbound variable" errors under `set -u` on macOS shell environments.
- Disabled `set -u` check globally in the governance script by changing `set -euo pipefail` to `set -eo pipefail` to ensure robust cross-platform git pre-push execution.
- Fixed observation primitive calculation logic in `WorkoutObservationPrimitiveBuilder` to return nil drift/stability and empty distribution when sample count is sparse/insufficient, matching snapshot expectations.
- Localized reason snippets and assertion strings to Chinese across core tests and sample cases to resolve suite-wide failures due to translation discrepancies.
- 填入真實 Strava 憑證（Client ID: 248735）至 `StravaAdapter.swift` 中的 `StravaCredentials` enum；Repo 為 private，憑證由使用者確認後直接導入。
- 修正 Strava OAuth "Invalid redirect URL" 錯誤：將 `redirect_uri` 從 `zonetruth://strava/callback` 改為 `zonetruth://localhost/strava/callback`（host 改為 `localhost`），對應 Strava API 設定頁的 Authorization Callback Domain 欄位應填 `localhost`。
- 將 Strava OAuth 授權流程從 `openURL + .onOpenURL` 改為 `ASWebAuthenticationSession`：在 app 內開啟 mini-browser，自動攔截 `zonetruth://` redirect，不再依賴 Safari 跳轉，修正 iOS 16+ Safari 不自動返回 app 的問題。
- 進一步簡化 redirect_uri 為 `zonetruth://localhost`（移除路徑），符合 StravaKit 等成熟 SDK 的慣例，修正 Strava 伺服器對帶路徑 custom scheme URI 的驗證拒絕問題。
